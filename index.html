What is an IP Address?

An IP address (Internet Protocol address) is a computer's unique address on the internet, like a house's unique address in the world.

Key Characteristics

  • Usually represented numerically like 198.51.100.7
  • 32 bits divided into four 8-bit groups
  • Each group written as decimal number (0-255)
198.51.100.7 in binary: 11000110.00110011.01100100.00000111 0 = 00000000 255 = 11111111

Why Domain Names?

IP addresses like 216.58.193.206 are hard to remember and can change. Domain names like google.com are easier to use!

Domain Name System (DNS)

The DNS process converts domain names to IP addresses:

  1. Computer wants to visit cnn.com
  2. Sends request to DNS server to lookup IP address
  3. DNS server returns IP address
  4. Computer can now communicate with CNN
Root Servers: 13 main DNS servers exist worldwide, maintained by organizations like Verisign, USC, University of Maryland, and NASA.

Domain Registration

  • Anyone can register unused domain names
  • Yearly fees required for registration
  • Information publicly available via ICANN's Whois service
  • Web hosting companies often manage DNS updates

Component Details

  • Scheme: Protocol (http, https, mailto, file)
  • Hostname: Complete domain name
  • Path: Specific resource location
  • Query String: Data for the server (after ?)
  • Fragment: Location within page (after #)
Browser Behavior: Modern browsers hide schemes and "www" for cleaner display but still use complete URLs for requests.
